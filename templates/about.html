Project Overview

Compiler Learning Hub is an interactive platform for students and enthusiasts to learn compiler theory, parsing algorithms, and language design. It provides tools like FIRST & FOLLOW calculator, Ambiguity Checker, LR(0) Parser, TAC generator, CFG visualizer, and more.

Features

  • FIRST & FOLLOW Calculator
  • LL(1) Parsing & Table Generation
  • Ambiguity Detection
  • Left Recursion & Left Factoring Tools
  • LR(0) Parser & DFA Visualization
  • Three Address Code (TAC) Generator
  • Regular Expression & Language Conversion Tools

Technologies Used

  • Frontend: HTML5, CSS3, JavaScript
  • Backend: Python (Flask)
  • Tools: GitHub, Git, Font Awesome
  • Responsive design with Flexbox & CSS Grid

Book Reference

"Compiler Blueprint: An Academic Guide to the Concepts of Compiler Design"
Mastering Concepts and Coding: Your Path to Academic Excellence
Authors: Md. Naimur Rahman & Sajjad Hossen Shemanto

This project complements the book by providing interactive tools, practical exercises, and visualizations to understand compiler design concepts more effectively.
